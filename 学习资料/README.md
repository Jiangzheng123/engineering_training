### 规范

- 前端和后端项目统一使用IDEA进行管理
	- IDEA版本为2024.2.3
	- 通过https://www.idejihuo.com/来获取激活码，方式一即可，到期了再续就行
- 要有良好的编程规范，在用到某些技术的地方将其用注释的形式描述出来
- 前后端要严格按照接口规范进行对接
